USAGE:

java -classpath .;simplestudy.jar;lib\log4j-1.2.15.jar org.ladder.collection.SimpleCollection [arg1] [arg2] [arg3] [arg4] [arg5]




ARGUMENTS:

[arg1] - location of the collection file (ex: coa_collection.txt)

[arg2] - boolean ("true" or "false") specifying if study should be random or not

[arg3] - location of the folder to save the data to (ex: data)

[arg4] - location of an image to display as the background (ex: map.png); user "." for no map

[arg5] - boolean ("true" or "false") specifying if the study should allow audio capture or not
